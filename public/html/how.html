How it works?

First of all, We'll need to create the html structure. See these two divs above, they will become the final hexagon

On this step, let to set the internal div (circle) shape using the clip-path css property

The next consists to apply Edge Shadow using the clip-path css property too in the ::after pseudo element css property

Apply the background shadow. It's a realy thorough detail applied to the edge shadow. In the final result, works like an extension

Animate it!

<div id="central-cell-shadow" class="central-cell-shadow"> // The black Square
   <div id="central-cell" class="central-cell"></div> // The white circle
</div>
.central-cell {
   width: 150px;
   height: 150px;
   margin: 0;
   border-radius: 0;
   clip-path: polygon(25% 0%, 75% 0%, 100% 50%, 75% 100%, 25% 100%, 0% 50%);
}
.central-cell::after {
   content: "";
   display: block;
   margin-left: -4px;
   width: 150px;
   height: 150px;
   filter: opacity(1);
   background-color: rgb(197, 197, 197);
   clip-path: polygon(25% 0%, 34% 0%, 8% 50%, 34% 100%, 25% 100%, 0% 50%);
}
.central-cell-shadow {
   filter: drop-shadow(-4px 0 5px rgba(0, 0, 0, 0.3));
}
